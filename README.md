# Cooperative UAV Path Planning: GTCPSO with HDPLO and Dynamic RHC

## Overview

Implementation of cooperative multi-UAV 3D path planning using:

1. Standard PSO (Cartesian baseline)
2. GTCPSO (cylindrical coordinates + gene targeting)
3. GTCPSO+HDPLO (adaptive crossover + CMA-ES + direction-guided search + diversity control)
4. Dynamic GTCPSO+HDPLO with RHC (online event-driven replanning)

The framework supports terrain, threat zones, no-fly zones, and inter-UAV safety constraints.

---

## Multi-Objective Cost

Mission objective:

J = w1*L + w2*T + w3*S + w4*A + w5\*F + P_constraints

| Component | Description                     | Weight |
| --------- | ------------------------------- | ------ |
| L         | Path length                     | 1.0    |
| T         | Threat exposure (piecewise)     | 5.0    |
| S         | Smoothness                      | 2.0    |
| A         | Altitude variation              | 0.5    |
| F         | Fuel consumption                | 0.3    |
| P_coll    | Inter-UAV collision penalty     | 200.0  |
| P_const   | Turn/climb constraint penalties | 50.0   |

## Steps To Run

### 1) Install dependencies

```bash
pip install -r requirements.txt
```

### 2) Generate datasets (optional, if you changed scenario generation)

```bash
python datasets/generate_datasets.py
```

### 3) Run quick static benchmark (all scenarios)

```bash
python experiments/run_fast.py
```

### 4) Run full benchmark suite (heavier, multi-run statistics)

```bash
python experiments/run_experiments.py
```

### 5) Run only simple case

```bash
python experiments/run_simple_case.py
```

### 6) Run dynamic demos

```bash
python experiments/demo_dynamic_run.py
python experiments/demo_showcase_run.py
python experiments/demo_pure_static_run.py
python experiments/demo_deviation_comparison.py
```

### 7) Generate dynamic scenario galleries

```bash
python experiments/generate_scenarios.py
python experiments/generate_large_scenarios.py
```
